# Contribution Guide

Hi! I am really excited that you are interested in contributing.
All Issues and PRs are welcome! Thank you!

## 👨‍💻 Setup

To setup:

```sh
npm ci
```

## 🧪 Develop

To test:

```sh
npm test run
```

To format:

```sh
npm run lint
```

To fix:

```sh
npm run fix
```

## 🚀 Release (Admin)

```
npm version $SEMVER
```

```
git push origin $TAG
```
